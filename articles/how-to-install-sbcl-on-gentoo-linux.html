如何在 Gentoo 下搭建 SBCL

12 Jan 2014

粗率的说， SBCL（全名为 Steel Bank Common Lisp 是目前较为活跃的 Common Lisp 编译器和运行环境。可扩展与多语言覆盖等功能受到了广大 Lisp 爱好者的青睐）

目前笨人在学习 Lisp 语言，也来玩个。但是我是用 Gentoo Linux 安装，遇到几个问题，先记录下来。

首先用 Gentoo 下的 portage 包管理器检查看看有什么选项和需求

sudo emerge --ask sbcl

但是安装之后会遇到 core file not found 的问题。稍微查了一点资料。需要几个环境变量。

touch /etc/env.d/50sbclsudo echo SBCL_HOME=/usr/lib/sbcl > /etc/env.d/50sbclsudo echo SBCL_SOURCE_ROOT=/usr/lib/sbcl/src >> /etc/env.d/50sbcl

然后更新下环境，更新环境变量

sudo env-update

安装完毕
